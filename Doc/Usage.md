# 指令集
```
    /dj (图片参数) (图片尺寸) (图片质量) (风格化) (图片种子) (图组差异化) (参考图片) (参考图片权重) -- 生成图片 [各版本要求不一致]
    /dblend (图片) (图片) (图片) (图片) (图片) (图片差异化) -- 融合图片
    /describe (图片) -- 描述图片
    /dsettings -- 调整DandJourney相关参数
    /dabout -- 关于DandJourney
    /dhelp -- DandJourney使用帮助
```
